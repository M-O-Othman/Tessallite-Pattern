# Tessallite Illustration Brand Brief

Status: active
Last meaningful update: 2026-05-09

Illustrations in this folder should follow the Tessallite brand identity from
`/media/oz/work/codex_work/codex-work/tessallite_brand_identity`.

## Palette

| Role | Color |
| --- | --- |
| Corporate Green | `#006C35` |
| Premium Gold | `#D4AF37` |
| Charcoal | `#333333` |
| White | `#FFFFFF` |
| Soft Mint | `#F2F7F4` |
| Accessible Border | `#8A94A6` |
| Neutral Border | `#CBD5E1` |

## Direction

- Enterprise, calm, precise, trustworthy.
- Clean light backgrounds, preferably White or Soft Mint.
- Use subtle isometric tile patterns as the primary visual motif.
- Use one Premium Gold highlight tile or accent to represent intelligent
  routing, acceleration, or the critical verification gate.
- Keep decorative graphics restrained.
- Avoid hype-heavy, neon, noisy, dark, or stock-like compositions.
- Avoid white text on gold.
