# Simple Structural Element Illustrations

Status: active
Last meaningful update: 2026-05-09

This folder contains simple flat Tessallite-branded illustrations for the four
structural elements of the Tessallite Pattern. These versions avoid 3D,
isometric depth, and separated text panels.

## Style

- Flat 2D diagrams.
- Minimal embedded labels.
- Tessallite palette: `#006C35`, `#D4AF37`, `#333333`, `#FFFFFF`,
  `#F2F7F4`, `#8A94A6`.
- No 3D or isometric perspective.
- No separate text panel.
