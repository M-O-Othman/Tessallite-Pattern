# Tessallite Pattern Illustrations

Status: active
Last meaningful update: 2026-05-09

This folder contains Tessallite-branded illustrations for the Tessallite
Pattern project. The retained assets are generated PNG illustrations with text
embedded directly in the image.

## Brand Brief

- [BRAND_BRIEF.md](BRAND_BRIEF.md)

## Generated Text Versions

The generated-text set bakes the labels directly into the generated image and
uses a reserved left text panel to prevent overlap with the visual artefacts.

- [generated-text/INDEX.md](generated-text/INDEX.md)

## Simple Structural Element Versions

The simple structural-element set uses flat 2D diagrams with minimal embedded
labels, no isometric depth, and no separated text panel.

- [structural-elements-simple/INDEX.md](structural-elements-simple/INDEX.md)

## Usage Notes

- Use the generated-text set for the broader framework slides.
- Use the structural-elements-simple set for the four individual structural
  elements.
- Preserve the Tessallite palette and calm flat style for future additions.
