# Documentation Index

Status: active
Last meaningful update: 2026-05-09

This is the L0 router for the documentation set. Start here when you do not
know where a document belongs. It intentionally lists domains, not every
document; domain indexes route to detailed inventories and nested indexes.

## Routing Rule

Use the shortest path:

1. If you know the exact file, open it directly.
2. If you know the domain, read that domain's `_INDEX.md`.
3. If you know neither, start here, choose the domain, then read its index.
