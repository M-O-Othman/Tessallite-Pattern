# Adoption Maturity Scorecard

Status: active
Last meaningful update: 2026-05-09

Use this scorecard to evaluate how strongly a project applies the Tessallite
Pattern. Score each item from 0 to 3.

Scoring:

- 0: absent
- 1: occasional or informal
- 2: mostly consistent
- 3: required and enforced
